# Customer Analytics: Purchase Amount Distribution by Customer Segment

**Contact:** 23f2005452@ds.study.iitm.ac.in

## Project Overview

This project delivers a professional-grade data visualization for OKeefe Predovic's retail client, analyzing purchase amount distribution across customer segments. The visualization is designed for executive presentations and strategic planning.

## Business Context

- **Client:** Major retail company
- **Purpose:** Quarterly business review presentation
- **Audience:** Executive team and board of directors
- **Focus:** Customer spending behavior analysis across segments

## Repository Contents

- `README.md` - Project documentation
- `chart.py` - Python script for generating the visualization
- `chart.png` - Generated boxplot (512x512 pixels)

## Technical Specifications

- **Visualization Type:** Boxplot using Seaborn
- **Dimensions:** 512x512 pixels
- **Libraries:** Seaborn, Matplotlib, Pandas, NumPy
- **Output Format:** PNG

## Customer Segments Analyzed

1. **Premium** - High-value customers with premium memberships
2. **Regular** - Standard customers with consistent purchase patterns
3. **Occasional** - Infrequent shoppers with sporadic purchases
4. **New** - Recently acquired customers

## Key Insights

The visualization reveals:
- Purchase amount distribution and variability across segments
- Median spending levels by customer type
- Outlier detection for unusual purchase patterns
- Statistical comparison of customer segment behavior

## Usage

```bash
# Install required libraries
pip install seaborn matplotlib pandas numpy

# Generate the visualization
python chart.py
```
